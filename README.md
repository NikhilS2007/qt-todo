# Qt To-Do App

A simple desktop to-do list application built with Python and PySide6 (Qt for Python).

## Requirements
- Python 3.7+
- PySide6

## Installation
1. Install Python.
2. Install PySide6:
	```powershell
	pip install PySide6
	```

## Running the App
1. Open a terminal in this folder.
2. Run:
	```powershell
	python todo.py
	```
